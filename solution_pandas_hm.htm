      - DataFrame is very useful structure to manipulate datas in pandas. Data frame is a way to store data in rectangular grids that can easily be overviewed.
      - You can specify the index and column names for your DataFrame. It is handy way to manipulate your data
      - with describe() command you can see a quick statistic summary of your data
      - Boolean indexing allows you to find necessary points
      - with one command -dataFrame.dropna()- data can be cleaned from rows with any column having null data
      - matplotlib package that is used to visualize data is usable in pandas
      - .apply() is used to apply a function to all columns
      - .map property applies changes to every element of a column. It is similar with .apply but you can do all changes in one line with .map
      - .unique() returns unique entries in selected columns. It allows us to see unique elements in a column
      - merging dataframes: essential way to collate informations come from different sources
      - pandas has easy sorting applications
      - Below commands are used to see summary of spesific data:
         - .sum : sum of values in a dataframe
         - .max : maximum value of a dataframe
         - .min : minimum value of a dataframe
         - .mean : average value of a dataframe
